## About Alba

Alba is PHP Framework for teaching purposes
- Simple, fast routing engine.
- Expressive, intuitive database ORM.

## Dependencies
- [PHP](https://www.php.net/) 8.1 or above
- [Composer](https://getcomposer.org/)
